## ✨ Project Overview

MeetForge is a web application engineered to **streamline meeting scheduling and enhance client communication**. It addresses the common pain points of manual coordination by automating Google Meet link generation and crafting personalized pitch emails. This project demonstrates proficiency in building efficient, user-centric solutions that integrate with external APIs to deliver tangible value.

## 🌟 Key Features & Impact

*   **Automated Google Meet Integration:** Eliminates manual setup, ensuring instant and professional meeting environments.
*   **Intelligent Pitch Email Generation:** Leverages advanced algorithms to create personalized, engaging outreach, significantly improving client engagement.
*   **Seamless Sender Identity:** Maintains brand consistency and trust by dispatching all communications directly from the host\'s connected Google account.
*   **Effortless Onboarding & UX:** Provides a frictionless sign-in experience (Google/Email), showcasing a focus on intuitive user interfaces and minimal setup overhead.

## 🛠️ How It Works (User Flow)

MeetForge simplifies the user journey into three core steps:

1.  **Sign In:** Quick authentication via Google or email, designed for immediate access without complex configurations.
2.  **Fill Details:** Users input client information (name, email, time), and the system intelligently drafts a tailored pitch.
3.  **Send & Relax:** A dedicated Google Meet link is generated, and the personalized pitch email is sent, allowing users to focus on their core tasks.
